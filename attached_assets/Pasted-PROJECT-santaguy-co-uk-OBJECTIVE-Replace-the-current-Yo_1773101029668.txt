TASK 4 — Play button styling

Style the play button similar to the YouTube play icon.

Requirements

• red circle
• white triangle
• centered
• subtle hover animation

Example CSS

.video-card {
 position: relative;
 cursor: pointer;
 overflow: hidden;
}

.video-card img {
 width: 100%;
 display: block;
}

.play-button {
 position: absolute;
 top: 50%;
 left: 50%;
 transform: translate(-50%, -50%);
 width: 68px;
 height: 48px;
 background: #ff0000;
 border-radius: 12px;
 display: flex;
 align-items: center;
 justify-content: center;
 color: white;
 font-size: 26px;
 transition: transform 0.2s ease;
}

.video-card:hover .play-button {
 transform: translate(-50%, -50%) scale(1.1);
}
